Requirements
============
Bug Report Classification Tool — ISE Coursework

The following software and dependencies are required to run the code.

Operating System
----------------
Windows, macOS, or Linux

Python Version
--------------
Python 3.8 or higher (tested on Python 3.14.3)

Required Python Libraries
--------------------------
Install all dependencies using the following command:

    pip install scikit-learn pandas numpy scipy matplotlib

Library versions used during development:
- scikit-learn    >= 1.0
- pandas          >= 1.3
- numpy           >= 1.21
- scipy           >= 1.7
- matplotlib      >= 3.4

No GPU or special hardware is required. All code runs on standard CPU.
